Temprest

Temprest is an 2D RPG inspired by Old-School games like Earthbound, it focuses on a nameless detective following a runaway bride case that soon reveals supernatural elements

This Project has won the Game Small Team 1st Place award in the Excellence in Computer Graphics Awards (ECGA) for 2025
